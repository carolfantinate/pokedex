# 📖 Pokédex com API

## 📌 Sobre o projeto

Este projeto é uma **Pokédex simples desenvolvida com JavaScript**, que permite buscar informações de um Pokémon digitando seu nome.
Os dados são obtidos através de uma API pública e exibidos dinamicamente na página.

O objetivo do projeto é **praticar o consumo de APIs externas, manipulação de dados em JSON e interação com o DOM** usando JavaScript.

---

## 🛠️ Tecnologias utilizadas

* **HTML**
* **CSS**
* **JavaScript**
* API pública de Pokémon

---

## 🌐 API utilizada

Os dados são obtidos através da API pública:

[PokéAPI](https://pokeapi.co/)

---

## ▶️ Como usar

1. Digite o **nome de um Pokémon** no campo de busca.
2. Clique no botão **Buscar** ou pressione **Enter**.
3. As informações do Pokémon aparecerão na tela.

Exemplos de Pokémon para testar:

* pikachu
* charizard
* bulbasaur
* squirtle

---

Este projeto foi criado com fins **educacionais**, para praticar:

* consumo de APIs externas
* requisições com `fetch()`
* manipulação de dados JSON
* atualização dinâmica do HTML com JavaScript
